# WebGlitch
A WebGPU program generator. 

## Requirements:
1. Gradle 8.9+
2. Node v20+

## Building:
```console
gradle build

npm install 
```

## Usage:
```console
./webglich.py
```

* You can also use the scripts `webglitch.sh` and `webglitch.bat` to integrate with other tools
